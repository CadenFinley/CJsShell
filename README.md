## Installation

### Homebrew Installation

```bash
brew install cadenfinley/cjsh/cjsh
```

### Manual Installation

You can also build and install from source:

```bash
# Clone the repository
git clone https://github.com/CadenFinley/CJsShell.git
cd CJsShell

# Build the project
mkdir build && cd build
cmake ..
make
```
